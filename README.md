# Zero Event Talks App

This is a simple single-page website for a one-day technical event. It displays a schedule of talks and allows users to filter them by category.

## How to run

1.  **Build the website:**
    ```bash
    node build.js
    ```

2.  **Serve the website:**
    Navigate to the `dist` directory:
    ```bash
    cd dist
    ```
    And run a simple python web server:
    ```bash
    python3 -m http.server
    ```

3.  **View the website:**
    Open your web browser and go to `http://localhost:8000`.
